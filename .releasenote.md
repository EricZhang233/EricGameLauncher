Eric Game Launcher (EGL) **v1.0.7**

- **智能化在线更新系统**
    - 每次启动时自动检测更新，确保版本信息始终与云端同步。
    - 检测逻辑全面优化：启动后延迟 3 秒触发，优先保障界面流畅性。
    - 新版本提醒更醒目：标题栏版本号变为红色，并附加同步图标。
    - 下载触发逻辑重构：主程序专注调用 Updater，下载与安装全权交由 Updater 处理。
    - 网络适配更灵活：GitHub API 失败时自动切换至中国大陆镜像站。

- **全新升级的更新器 (Updater)**
    - 分阶段任务系统：从下载到重启，五步流程一气呵成。
    - 下载进度实时可视化：借助 `Spectre.Console` 呈现动态进度条。
    - 数据保全机制：更新过程中严格保护 `data` 文件夹，其余文件智能清理。
    - 自动提权功能：目录权限不足时，自动弹出管理员授权请求。

- **技术细节**
    - Updater 资源化嵌入：用户确认更新后自动释放至临时目录。
    - 构建流全面精简：
        - 移除外部脚本，改用 MSBuild 原生函数同步版本号。
        - 清理解决方案时强制删除所有中间文件，确保物理磁盘整洁无残留。
        - 发布路径更直观：安装包直接存放于 `output/` 根目录。
