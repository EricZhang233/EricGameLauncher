Eric Game Launcher (EGL) 迎来深度优化版本 **v1.0.1**！

本次更新重点提升了属性配置面板的交互便捷性与视觉统一性，并修复了关键的调试兼容性问题。

## 更新亮点

### 🎨 UI 与交互重构
*   **标题样式统一**：属性面板的所有模块标题（主程序、代替启动、同时启动、管理器）现在拥有统一的字体大小与粗细，视觉重心更清晰。
*   **控件布局对齐**：将复选框文字与标题分离，解决了复选框自带文字无法精准控制样式的问题，界面更加整洁。
*   **快捷选择全覆盖**：原本仅主程序路径拥有的“开始菜单/桌面/浏览”快捷选择功能，现在已全面覆盖至“代替启动”和“同时启动”模块。
    *   *场景*：您可以更快速地配置 Starward 代替启动，或一键添加游戏翻译助手作为伴随启动程序。

### 🚀 商店应用支持优化
*   **开始菜单列表重构**：重构了应用扫描逻辑，现在开始菜单应用会以平铺列表显示，查找更直观。
*   **商店应用全兼容**：完善了对通过 Microsoft Store 安装的应用（如《极限竞速：地平线 5》）的图标提取与管理员模式启动支持。

### 🛠️ 稳定性修复
*   **调试器兼容性**：修复了在 Visual Studio 调试模式下由于目录结构变动导致的 `0xe0434352` 启动崩溃问题。
*   **配置管理**：修复了设置页面中“迁移模式”按钮可能出现的错误鼠标提示。

## 快速开始

1.  **环境准备**：确保电脑已安装 [.NET Desktop Runtime 8.0](https://dotnet.microsoft.com/en-us/download/dotnet/8.0)。
2.  前往 [Releases](../../releases) 页面下载最新版本。
3.  解压并运行 `EricGameLauncher.exe`。
4.  *(推荐)* 在设置中开启“便携模式”，体验绿色软件的快乐。
