# Eric Game Launcher (EGL) **v1.0.10.1**  稳定版-修复

> [!IMPORTANT]
> **此版本为稳定版**

> [!NOTE]
> 每个版本号第三位为[10]的倍数的版本(如x.x.10、x.x.20)将被视为稳定版，稳定版可能存在第四位版本号(如x.x.10.x)。

### Bug 修复与稳定性强化 (Bug Fixes & Improvements)

- **修复管理员权限启动 UWP 应用时的黑框闪烁问题**
  优化了使用管理员权限启动系统及 UWP 商店应用时的 PowerShell 脚本调用逻辑，通过显式禁用 Shell 启动并强制应用无窗口模式，实现了提权操作的完全静默，不再出现弹出的控制台黑框。

- **修复目标程序运行目录 (Working Directory) 继承错误**
  完善了程序拉起各类游戏、应用以及内部更新组件（ConfigUpdater / MainUpdater）的逻辑。在拉起子进程时已实现了精确的工作目录 (Working Directory) 指派，确保游戏和应用运行在自身的真实路径下。此修复彻底解决了部分游戏侦测到继承了启动器目录而导致的路径异常、DLL 读取缺失、防作弊软件误判以及依赖自身环境运行出错等兼容性问题。

- **修复“运行管理器”等动态右键菜单点击无效的问题**
  修复了在游戏项目的右键菜单中，动态生成的“运行管理器”等选项因上下文数据绑定异常而导致点击后无响应的问题。

- **修复界面部分文案的国际化失效问题**
  修复了在添加/编辑自定义右键菜单时，“浏览文件...”子菜单的显示文本未正确匹配多语言配置从而直接显示回退键名的问题。

- **优化“检查更新”逻辑，新增“修复”功能**
  增强了应用内手动检查更新的功能：在当前已为最新版本时，现在也会向您展示当前版本的详细更新日志。同时弹窗中新增了“修复”选项，允许用户在遇到文件损坏或特定异常时，强制触发当前版本的重新下载与覆盖安装。
