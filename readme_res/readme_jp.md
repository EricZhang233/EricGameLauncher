## はじめに

**Eric Game Launcher** は、ミニマリストのために作られた次世代のゲームランチャーです。従来のプラットフォームの肥大化を排除し、**WinUI 3** と **.NET 8** の強力なパフォーマンスを活用することで、ミリ秒単位の起動体験と Windows 11 ネイティブの視覚的な楽しみを提供します。

ゲームが Steam、Epic、または独立した exe ファイルであっても、ここがそれらの統一された拠点となります。

## 機能特性

### プレイヤーのために
*   **究極の軽量化**：必要な時に起動し、使い終わったらすぐ終了。バックグラウンドサービスなし、広告なし、システムリソースの使用率はほぼゼロです。
*   **効率的な検索**：英語のイニシャル、スペース区切りの頭文字、およびキャメルケース（CamelCase）マッチング検索をサポートしています。「League of Legends」をプレイしたいですか？「LOL」と入力するだけで、結果が即座に表示されます。
*   **プロフェッショナル級のプロパティ制御**：これは単なる簡素なショートカットではありません。
    *   **管理者として実行**：「ゲームメインプログラム」と「ゲームマネージャー」（Steam/Epic など）に対して個別に権限昇格を設定でき、権限不足で起動できない問題を解決します。
    *   **マネージャーの実行**：マネージャーのパスを設定することで、ゲームが起動するだけでなく、プラットフォームサービスも正しく呼び出されるようにします。
    *   **フルパラメータサポート**：メインプログラム、マネージャー、代替起動、およびコンパニオンプログラムのすべてのパスで、**コマンドライン引数**または **URL プロトコル**をサポートしています。
    *   **メインプログラムの代替起動**：この項目をチェックすると、カスタムコマンド（例：`starward://`）を使用して、元の EXE の起動ロジックを完全に置き換えることができます。
        > 💡 **ヒント**：EGL でゲームカバーを保持したまま、実際には外部ツール（Starward など）を呼び出してゲームを起動し、プレイ時間を記録できます。
    *   **起動時に同時実行**：ゲームをプレイする時に翻訳機、タイマー、またはパフォーマンスモニターを自動的に開きたいですか？これを設定すれば、ワンクリックで同時起動が可能です。
    *   **アイコンの変更**：カバーとして手動で画像をアップロードすることをサポート。
    *   **カスタム右クリックメニュー**：各プロジェクトに最大 10 個のカスタムメニュー項目を追加可能。
        *   タイトル、コマンドパス、引数を個別に設定可能。
        *   カスタム項目ごとに「管理者として実行」を設定可能。
        *   **スマート位移**：中間項目を削除すると、後続の項目が自動的に繰り上がり、リストをコンパクトに保ちます。
    *   **アイコンサイズ**：コンパクトなリストから没入感のある大きなタイルまで、自由自在に変更できます。

### ギークのために
*   **完全ポータブルモード (Portable Mode)**：すべてのデータ（設定、キャッシュされたアイコン）をプログラムディレクトリ内に保存することをサポート。USB メモリに入れて、あなたのゲームライブラリを持ち運び、どの PC でも実行できます。
*   **シームレスなデータ移行**：システムインストールモードからポータブルモードへの切り替えも、ワンクリック移行ですべての設定とアイコンが自動的に移動し、再設定は不要です。
*   **スマートなアイコン抽出**：`.exe`、`.lnk`、さらには Steam/Epic の URL プロトコルを自動的に解析し、高解像度のアイコンを抽出してキャッシュします。

## 技術的なハイライト

開発者にとって、Eric Game Launcher は最小限のコードでいかに現代的な Windows デスクトップアプリを構築するかを示しています。

*   **最先端の技術スタック**：最新の **Windows App SDK (1.6+)** と **.NET 8** に基づいて構築されており、デスクトップアプリにおける WinUI 3 の大きな可能性を示しています。
*   **ネイティブパフォーマンス**：
    *   `P/Invoke` (User32.dll) を利用して、効率的なシステムレベルのアイコン抽出を実現。
    *   `System.Text.Json` を使用して、高性能でメモリ使用量の少ないデータシリアル化を実現。
*   **クリーンなアーキテクチャ**：
    *   **効率的なデータバインディング**：`ObservableCollection` と XAML バインディングにより、スムーズな UI インタラクションと明快で直感的なコードロジックを実現。
    *   **静的サービス設計**：`ConfigService` や `I18n` などのコアモジュールは静的クラスとして設計されており、オーバーヘッドゼロで呼び出せ、極めてシンプルかつ効率的です。
*   **モダンなビルド**：
    *   **Unpackaged リリース**：煩雑な MSIX パッケージングや証明書のインポートを廃止し、クリーンな `.exe` 実行ファイルを直接生成します。
